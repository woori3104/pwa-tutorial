
# What is Progressive web 

- 모바일 앱과 같은 경험을 주는 최신 웹 앱
- 앱 아이콘, 앱 아이콘 설치 배너, 모바일 푸시, 오프라인 경험을 제공
- Html, css, js로 구현하는 웹 애플리케이션 

# 프로그레시브 웹 앱의 등장 배경
- 모바일 시장의 폭발적인 성장
- 모바일 앱영역을 커버하기 위한 시도 
  - hybrid App, React Native
- Offline Web의 필요성

# 프로그레시브 웹 앱의 특징
- Responsive 
  - 반응형 웹 디자인 
- App-like & Discoverable
  - 특정 브라우저에서 url을 이동시 다운로드 가능
  - 앱과 같은 형태로 접근가능
    - 앱스토어나 구글스토어에서의 설치x
- Engageable
  - PC와 모바일 단말기 모두 커버가능 
- Connectivity
  - Offline에서도 사용가능 
- Safe
  - https를 이용해서 구축해야함
